# 徒手训练周计划

**徒手训练的核心原则：**
既然没有了外部重量，我们就需要用其他方式来“上强度”以刺激肌肉：

1.  **增加次数 (Reps):** 将每组次数提高到 15-25 次，甚至力竭。
2.  **控制节奏 (Tempo):** 放慢动作，尤其是“离心”阶段（例如：俯卧撑“下去”的过程用 3 秒）。
3.  **减少组间休息 (Rest):** 将休息时间缩短到 45-60 秒，增加代谢压力。
4.  **增加难度变式 (Variation):** 例如，用“单腿”动作（单腿臀桥）代替“双腿”动作。
